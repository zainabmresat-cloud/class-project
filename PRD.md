# Product Requirements Document (PRD)

## Goal
Build a simple API that classifies text into categories.

## Features
- Accept text input
- Return classification category
- Provide confidence score

## API Endpoints
GET / → server status  
POST /classify → classify text

## Technical Stack
- Node.js
- Express
- Jest (testing)

## Non-Functional Requirements
- Fast response time
- Simple and clean code
- Easy to test
